# Chatbot Project

## Setup Instructions

1. Install Python and Anaconda.
2. Create a virtual environment with Anaconda:
    ```bash
    conda create -n chatbot-env python=3.8
    conda activate chatbot-env
    ```
3. Install the required libraries:
    ```bash
    pip install -r requirements.txt
    ```
4. Run the Flask app:
    ```bash
    python run.py
    ```
